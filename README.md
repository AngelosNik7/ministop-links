# Mini Stop Express Links

Static one-page website for GitHub Pages.

## GitHub Pages setup

1. Create a new GitHub repository named `ministop-links` under username `Angelosnik7`.
2. Upload `index.html` from this folder.
3. Go to Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: `main` / root.
6. Save.

Your website will be:

https://Angelosnik7.github.io/ministop-links/
